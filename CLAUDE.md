# CLAUDE.md

## Product Vision

CareerFlow helps job seekers organize and optimize their job search using AI.

The platform combines:

* Resume analysis
* ATS optimization
* Application tracking
* Interview management
* Analytics

into one modern SaaS application.

---

## Target Users

Primary:

* Students
* Freshers
* Developers
* Software Engineers

Secondary:

* Career changers
* Professionals seeking new jobs

---

## Core User Goals

Users should be able to:

1. Upload resumes
2. Receive AI-powered feedback
3. Improve ATS compatibility
4. Track applications
5. Track interviews
6. Monitor progress through analytics

---

## Main Features

### Authentication

Users can:

* Register
* Login
* Logout

Providers:

* Google
* GitHub
* Credentials

---

### Resume Management

Users can:

* Upload resumes
* View uploaded resumes
* Delete resumes
* Analyze resumes

Supported format:

* PDF

---

### AI Resume Analysis

AI should provide:

* ATS score
* Missing keywords
* Formatting suggestions
* Improvement recommendations
* Skill gap analysis

Analysis should be actionable.

Avoid generic responses.

---

### Job Tracker

Users can manage:

* Company
* Role
* Status
* Salary
* Application Date
* Notes

Statuses:

Applied

Online Assessment

Interview

Rejected

Offer

Accepted

---

### Analytics Dashboard

Dashboard should visualize:

* Total applications
* Interview conversion rate
* Offer rate
* Monthly activity

Charts should be simple and useful.
